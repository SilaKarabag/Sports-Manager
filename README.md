SPORTS MANAGER - Installation Instructions
==========================================

Requirements:
  - Java 11 or newer (JRE or JDK)
  - Windows 10 / Windows 11

To run:
  1. Double-click run.bat
     OR
  2. Open Command Prompt here and type:
     java -jar sports-manager-1.0-SNAPSHOT.jar

To compile from source (requires Maven + JDK):
  mvn clean package
  Then run as above.

Save files:
  - Save games are .dat files
  - You can save anywhere on your computer
  - Load via Main Menu or in-game menu
